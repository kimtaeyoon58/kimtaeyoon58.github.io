# #문제
~~~
비밀번호를 잊어버린 흥덕이는 회원가입 때에 등록한 세 가지 질문에 모두 정확한 대답을
하여야 임시 비밀번호를 발급받을 수 있다. 질문을 하고 답변을 입력받은 후 사용자 진위를
판단하는 프로그램을 작성하시오. 단, 흥덕이가 시스템에 등록해 놓은 질문과 답변은 다음과 
같다.
~~~
# #해결 
~~~
x = input('내가 졸업한 초등학교 이름은? :')
y = input('엄마 이름은?:')
z = input('좋아하는 가수는?: ')
a = '옥성초등학교'
b = '안지연'
c = 'BTS'
print(x == a and y == b and z == c)
~~~
# #결과 
~~~

엄마 이름은?:안지연
좋아하는 가수는?: BTS
True
~~~
